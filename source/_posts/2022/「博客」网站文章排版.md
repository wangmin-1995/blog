## <center>小标题

- ***「有序列表」***用于步骤说明

- ***「无序列表」***用于每一段落首行，代替必要的缩进

- ***「任务列表」***用于同级选项，选中为必备选项，未选中为可选选项

  {%tabs x%}

  <!--tab 小标题-->

  ### 「有序列表」

  ### 「无序列表」

  ### 「任务列表」

  <!--endtab-->

  <!--tab 有序列表-->

  {% label 用法 blue %}

  ~~~markdown
  1. 上班
  2. 摸鱼
  3. 下班
  ~~~

  {% label 效果 blue %}

  1. 上班
  2. 摸鱼
  3. 下班

  <!--endtab-->

  <!--tab 无序列表-->

  {% label 用法 blue %}

  ~~~markdown
  - 一只狗
  - 一只猫
  ~~~

  {% label 效果 blue %}

  - 一只狗
  - 一只猫

  <!--endtab-->

  <!--tab 任务列表-->

  {% label 用法 blue %}

  ~~~markdown
  - [ ] 好好工作
  - [x] 好好摸鱼
  ~~~
  
  {% label 效果 blue %}
  
  - [ ] 好好工作
  - [x] 好好摸鱼

  <!--endtab-->
  
  {%endtabs%}
