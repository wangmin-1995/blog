网站信息

  • 「Site」

网络地址

  • 「Url」

「文章永久链接」

  • 「permalink」

    默认设置，文章在「GitHub」仓库中的目录是年/月/日/文章名/index.html

目录

  • 「Directory」

文章

  • 「Writing」

「新文章路径」

  • 「New Post Name」

    hexo n 新文章时，新文章生成的目录位置

    本地路径：source/_posts/文章名/文章名.md

    本地路径：source/_posts/年/月/日/文章名/文章名.md

「草稿」

  • 「Draft」

    Hexo 的一种特殊布局：「draft」，这种布局在建立时会被保存到 source/_drafts文件夹，通过 「publish」命令将草稿移动到 source/_posts 文件夹

    草稿默认不会显示在页面中，在执行时加上「–draft」参数，或是把「render_drafts」参数设为「true」来预览草稿。

元数据元素

  • 「Metadata Elements」

    是否在页面开头插入下面的meta标签，默认为「true」
